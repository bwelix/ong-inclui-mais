# ONG Inclui +

## Sobre o Projeto

A ONG Inclui+ é uma plataforma web desenvolvida para conectar organizações do terceiro setor com voluntários, doadores e recursos necessários para ampliar seu alcance e maximizar seu impacto social.

## Objetivo do projeto

Democratizar o acesso à tecnologia de gestão para organizações do terceiro setor, permitindo que se concentrem no que realmente importa: causar impacto positivo na sociedade.

## Funcionalidades

### Para ONGs

• **Gestão de Projetos:** Planeje, execute e acompanhe todos os seus projetos

• **Gestão de Voluntários:** Conecte-se com pessoas dispostas a fazer a diferença

• **Divulgação de Projetos:** Conte sua história para o mundo e amplie seu alcance

• **Captação de Recursos:** Diversifique suas fontes de financiamento

###  Para Voluntários

• Encontre oportunidades que combinam com seu tempo e habilidades

• Flexibilidade de horários e modalidades de trabalho

• Certificação de horas voluntárias

• Conexão com uma comunidade de agentes de transformação

### Para Doadores

• Transparência total sobre os projetos apoiados

• Acompanhamento do andamento das iniciativas

• Relatórios de impacto das doações

• Diversas modalidades de apoio

## Tecnologias Utilizadas

• **Frontend:** HTML5, CSS3, JavaScript (ES6+)

• **Design:** Layout responsivo, animações CSS, UX/UI moderno

• **Funcionalidades:** Validação de formulários, máscaras de entrada, navegação suave

## Como Usar

### Para ONGs

1. Acesse a página de cadastro

2. Selecione "Sou uma ONG"

3. Preencha os dados da organização

4. Aguarde o contato da nossa equipe (até 48h) 

### Para Voluntários

1. Acesse a página de cadastro

2. Selecione "Sou Voluntário"

3. Informe suas habilidades e disponibilidade

4. Comece a receber oportunidades compatíveis

###  Para Doadores

1. Navegue pela plataforma

2. Conheça as organizações cadastradas

3. Escolha a causa que deseja apoiar

4. Faça sua doação com transparência
